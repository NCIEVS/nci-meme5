UMLS Terminology Browser

This demonstration UMLS terminology browser and app is loaded with a Level 0 + SNOMEDCT_US edition of the 2018AB UMLS® Metathesaurus. It contains complete versions of many terminologies, including SNOMED CT, MeSH, ICD10, RXNORM, MED-RT, and others. The data set contains over 1.6 million concepts with over 5.5 million individual terms from constituent terminologies.

Fast and Convenient Searching

Quickly and easily search for concepts, descriptors, and codes with highly tuned algorithms based on Apache Lucene indexing. Search features include:
  • Simple and intuitive text entry to search by name or code
  • Typeahead (autocomplete) for textual searches
  • Scored results indicate fidelity of matching terms.
  • Spelling correction for commonly misspelled terms.
  • Comprehensive acronym expansion.
  • Automatic pattern matching for chemical structure and other complex names.
Use advanced options to restrict and refine your search by language, semantic and term types, or terminologies within a metathesaurus.
 
 
 

Navigate Hierarchical Terminologies

Explore hierarchical terminologies and view multiple paths to your search results. Our hierarchical tree view allows you to:
  • Browse a terminology from the top-down, and inspect objects as you go.
  • Retrieve search results as hierarchical trees instead of a simple list.
  • Open and close parents, siblings, and children for any object.
  • Highlight and retrieve details for terminology extensions or subsets.
  • View and browse all individual paths to any given object.
 
 
 

Metathesaurus Deep Relationships

Browse through "deep" relationships between concepts in a metathesaurus terminology (e.g. UMLS). Search through deep relationships by text, or sort by terminology connection, relationship type, and target object details.
  • Extrapolate source-asserted relationships up to the Metathesaurus level.
  • High-performance, supporting concepts with many thousands of relationships.
 

Easy Navigation and Session History

Explore concepts in the Metathesaurus as well as from the perspective of individual source terminologies without getting lost.
  • Navigate between contents and terminologies with convenient in-line links.
  • Quickly return to previously viewed content using the session history controls at the top of the content report.
  • "Pop out" views into new windows/tabs for side-by-side comparison.
 
 

Save Favorite Content

Maintain user-specific favorites list for content across all terminologies in the UMLS library. Add and remove favorites from any point in the application. Filter and sort your personal list.
 

Annotate Content

Annotate content from reports or from your favorites list with rich-text and HTML formatting support.

  • See other user's notes and reply to their comments.
  • Use advanced search features to retrieve content with specific annotations.
  • Quickly access annotations from the sidebar
 
 

Inspect Terminology Metadata

Immediately switch between viewing content and metadata with a click of a tab. Inspect complete lists of abbreviations and meanings for:
  • Precedence lists.
  • Attribute names
  • Relationship and additional relationship types
  • Languages
  • Semantic Types
  • Term Types
 
 

Customizable Features

The Terminology Server comes with easily overridden defaults for customizable display and function.
  • Rebrandable with logos, header/footer text, copyright, and links
  • Create security handlers for your own particular needs.
  • Customize search handling for particular terminologies or cases.
  • Write custom loaders for unsupported data formats, or tweak the existing loaders for standard formats (Owl, RRF, RF2, ClaML), etc.).
  • Create your own landing page, require a custom user license, display a custom login page.
